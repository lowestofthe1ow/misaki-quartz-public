Proposition Â§

A proposition is a declarative sentence that is either true or false, but not both. All the following declarative sentences are propositions; propositions 1 and 3 are true, whereas 2 and 4 are false.

  1. Washington, D.C., is the capital of the United States of America.E
  2. Toronto is the capital of Canada.
  3. 1 + 1 = 2.
  4. 2 + 2 = 3.

Propositional variables Â§

We use letters to denote propositional variables (or sentential variables), that is, variables that represent propositions, just as letters are used to denote numerical variables.

When declaring propositional variables, use simple (atomic) propositions. Avoid operators in declarations.

Truth value Â§

The truth value of a proposition is true, denoted by T, if it is a true proposition, and false, denoted by F, if it is a false proposition.

Formed from existing propositions using logical operators

Operators that take existing propositions to form compound propositions

Truth table Â§

A table consisting of n input columns, an output column, and xn rows, where x is the number of possible values (true or false means 2 values) and n is the number of variables
